# 助盲跑 App 使用手册

面向对象：**盲人用户**、**志愿者用户**。本手册只讲"怎么用"，不涉及技术实现（技术细节见 `frontend-guide.md`）。

---

## 一、产品是什么

助盲跑是一个连接**视障人士**和**志愿者跑者**的陪跑服务 App。盲人用户发起陪跑需求，系统自动匹配附近合适的志愿者，志愿者上门陪同完成跑步/出行，全程可实时定位、一键求助。

首次使用需要选择身份：**盲人** 或 **志愿者**，两种身份的功能和后续认证要求不同。

---

## 二、盲人用户功能

### 1. 注册登录
手机号 + 短信验证码登录，无需设密码。首次登录后选择"盲人"身份。

### 2. 发起陪跑订单
- 支持**立即出发**或**预约**（预约需提前至少 30 分钟下单，太临近会提示"预约时间太近"）
- 填写起点、终点、期望时间
- 提交后系统自动在附近志愿者中匹配，无需自己找人

### 3. 订单进度
下单后会经历以下阶段，App 内会实时提示当前状态：

| 阶段 | 含义 |
|---|---|
| 匹配中 | 系统正在附近寻找合适志愿者 |
| 待出发 | 志愿者已接单，准备出发 |
| 志愿者已出发 | 志愿者正在赶来的路上（可在地图看志愿者实时位置） |
| 志愿者已到达 | 志愿者已到达你的位置 |
| 陪跑中 | 志愿者点击"开始服务"后，正式进入陪跑 |
| 已完成 | 服务结束 |

如果长时间无人接单，系统会自动扩大搜索范围重新匹配；确实无人接单会提示"暂无志愿者"，可重新发起。

### 4. 紧急求助（SOS 按钮）
遇到危险或不适，随时按下 App 内的求助按钮，无需依赖是否有进行中的订单：

1. 按下后立即通知当前的陪跑志愿者（若有），同时后台客服也会收到告警
2. 志愿者会在收到提示后确认"误触"或"确实需要帮助"
   - 志愿者若 30 秒内没有响应，系统视为紧急情况自动升级
3. 一旦判断为真实求助，系统会**自动短信通知你的紧急联系人**，短信包含你的姓名、时间和大致位置（无法定位时会提示对方直接报警 110）
4. 客服会介入跟进直至标记"已解决"

⚠️ 求助有 60 秒冷却时间，避免误触连续触发。

### 5. 紧急联系人管理
- 最多可添加 5 位紧急联系人，需指定 1 位"主要联系人"（求助短信优先发给 TA）
- 支持增删改，本人可查看联系人的真实手机号（其他人无法通过 App 看到）

### 6. 查看志愿者位置
订单进入"志愿者已出发/已到达"阶段后，可在地图上看到志愿者的实时位置。

### 7. 完成后评价
服务结束后可为志愿者评分，评分会影响该志愿者后续被系统推荐匹配的优先级。

## 三、志愿者用户功能

### 1. 注册认证（三步）

| 步骤 | 内容 |
|---|---|
| 第 1 步 | 填写基本信息 + 身份证号，系统自动做身份证二要素核验（姓名+身份证号），失败会直接拦截，不能进入下一步 |
| 第 2 步 | 人脸活体认证（眨眼/点头等动作），确认"人证合一" |
| 第 3 步 | 完成 3 门必修培训课程 + 课后测验（60% 及格，可无限重考取最高分） |

⚠️ **当前产品阶段说明**：第 3 步的培训/测验目前**不作为接单门槛**——只要注册了账号即可被系统纳入派单候选并接单，不要求先完成培训课程或通过身份证/人脸认证。这是"先上线再说"的临时产品决策，后续可能恢复强制门槛，届时以 App 内实际提示为准。

培训课程本身仍有防作弊限制（按顺序学、进度条限速、题目乱序），只是完不完成不影响接单。

### 2. 接单流程（智能派单，无需抢单）
- 系统会按**距离、时间是否合适、历史评分、历史接单率、配速**综合打分，把订单派给最匹配的志愿者
- 每次只派给**一位**志愿者，你有 **30 秒**决定接受或拒绝
- 不响应算超时，系统自动派给下一位候选人（不会因此拉低你的"接单率"指标，但会记录"超时次数"）
- 可在"我的资料"里关闭"可服务状态"，关闭后不会再收到新派单

### 3. 服务流程
1. 接单后点击"出发"
2. 到达盲人位置后点击"已到达"
3. 点击"开始服务"正式进入陪跑
4. 陪跑结束点击"完成服务"

### 4. 取消 / 应急
- 志愿者中途取消，订单不会直接作废，会自动转入"重新匹配"给其他志愿者
- 陪跑过程中若盲人触发紧急求助，你会收到告警，需要确认"误触"或"需要帮助"；30 秒不处理会自动按严重情况升级

## 四、常见问题 FAQ

**Q: 预约陪跑最早能提前多久下单？最晚呢？**
A: 至少提前 30 分钟下单，太临近当前时间会被拒绝。

**Q: 志愿者不认证能接单吗？**
A: 目前**可以**——身份证核验/人脸活体/培训测验暂未作为接单硬性门槛（产品临时决策，先上线再说），后续可能恢复强制要求。

**Q: 求助按钮误按了怎么办？**
A: 志愿者端会先弹出确认，选择"误触"即可直接解除，不会打扰到紧急联系人。

**Q: 没设置紧急联系人时求助会怎样？**
A: App 会明确提示"未能通知到紧急联系人"，不会让你误以为已经通知到家人，同时转由客服直接跟进。

**Q: 志愿者定位多久更新一次？**
A: 志愿者出发后持续上报位置，盲人端地图基本实时可见（缓存有效期 30 秒）。
